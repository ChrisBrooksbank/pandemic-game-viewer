# PandemicGameViewer

An Angular 13 application for viewing Pandemic game state.

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 13.2.6.

## Prerequisites

- Node.js
- npm
- Angular CLI 13

## Development server

Install dependencies, then start the development server:

```bash
npm install
npm start
```

Navigate to `http://localhost:4200/`. The app reloads automatically when source files change.

## Build

```bash
npm run build
```

The build artifacts are written to `dist/`.

## Tests

```bash
npm test
```

Unit tests run via [Karma](https://karma-runner.github.io).

## Angular CLI

Use `ng generate component component-name` to generate components, or `ng help` for the Angular CLI command reference.
